# Welcome to React UX4G

**React UX4G** is a comprehensive React component library built upon the **User Experience for Government (UX4G)** design system. UX4G is an initiative by the Government of India (MeitY & NIC) designed to standardize and elevate the user interface, accessibility, and user experience across government portals and digital applications.

By bringing the official UX4G design system to React, this library enables developers to rapidly build modern, compliant, and accessible web applications that align with national digital standards.

## Features
- **40+ Components:** From simple buttons to complex data tables, overlays, and charts.
- **Accessible:** Built with standard WAI-ARIA guidelines in mind to ensure your apps are usable by everyone.
- **Customizable:** Easily themeable using standard CSS variables and utility classes.
- **Developer Experience:** Fully documented with comprehensive JSDoc IntelliSense support and detailed Storybook guides.
